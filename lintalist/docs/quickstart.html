Lintalist Quick Start Guide

  1. Once you close this window you can start adding and searching for your favourite Snippets by pressing [quickstarthotkey]. (You can change this)
  2. Once the Lintalist window is active you can add new Snippets to your bundle by pressing F7 - see the Edit menu for more options.
  3. To edit a Snippet: highlight a snippet - by searching and using the Up & Down arrow keys - and press F4.
  4. You can access various options (and change the hotkeys) via the Configuration options available via the Edit menu or tray menu (right click).
Important hotkeys
Start search: [quickstarthotkey]
Start Quick search: [quicksearchhotkey] Read docs first.
Quit Lintalist: [quithotkey]

Be sure to read the documentation to learn about how to use plugins and setup (multiple) context senstive bundles and other configuration options.
